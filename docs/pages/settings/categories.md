# Category Manager

## What is it?
**Category Manager** allows you to create custom labels to organize your data. You can create categories for **Contacts** (e.g., "VIP", "Lead") or **Products** (e.g., "Socks", "Shoes").

## Why is it useful?
- **Filtering**: Quickly find all "VIP" customers or all "Summer" products.
- **Reporting**: See analytics broken down by category (e.g., "Which product category sells the most?").
- **Visuals**: Assign colors and icons to tags to make them easily recognizable in the list view.

## Option Buttons (UI Guide)
- **Target Module**: Select if this tag applies to `Contacts`, `Products`, or `All`.
- **Name**: The label text (e.g., "New Lead").
- **Color**: Pick a tag color (e.g., Red for "Urgent").
- **Icon**: An optional emoji or icon.
- **Toggle Status**: Enable or disable a tag without deleting it.

## Use Cases
1.  **Lead Scoring**: You create tags "Cold", "Warm", and "Hot". Your sales team updates the contact tag after every call.
2.  **Inventory**: You tag products by season ("Winter Collection"). At the end of winter, you disable the category.

## How to Use
1.  **Navigate**: Go to **Settings** > **Categories**.
2.  **Add**: Click **New Category**.
3.  **Define**: Enter Name "VIP", Color "Gold", and select Target "Contacts".
4.  **Save**: Click **Create**.
